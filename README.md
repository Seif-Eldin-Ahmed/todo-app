# 📝 Todo List App

A simple and responsive Todo List web application built with HTML, CSS, and JavaScript.  
This project helps users manage their daily tasks efficiently with a clean and intuitive interface.

---

## ✨ Features
- ➕ Add new tasks
- ❌ Delete tasks
- ✔️ Mark tasks as completed
- 💾 Save tasks in browser using Local Storage
- 📱 Fully responsive design for all devices
- 🎯 Clean and user-friendly interface

---

## 🛠️ Built With
- HTML5  
- CSS3 (Flexbox / Grid / Responsive Design)  
- JavaScript (Vanilla JS)
